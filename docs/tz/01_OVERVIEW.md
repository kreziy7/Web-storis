## Что это

**Smart Reminder PWA** — production-ready прогрессивное веб-приложение для управления напоминаниями с полноценной offline-first архитектурой, двунаправленной синхронизацией данных и push-уведомлениями.

## Ключевая идея

Система спроектирована так, чтобы работать **без интернета**, накапливать изменения локально в IndexedDB и автоматически синхронизироваться с сервером при восстановлении соединения.

Пользователь **не должен замечать разницы** между работой онлайн и оффлайн.

## Пользователи системы

- Физические лица, управляющие личными напоминаниями
- Устройства: мобильные телефоны, планшеты, десктопные браузеры

## Тип приложения

| Параметр | Значение |
|---|---|
| Тип | Full-stack PWA |
| Frontend | React SPA, устанавливается как нативное приложение |
| Backend | REST API, Node.js + Express |
| База данных | MongoDB (сервер) + IndexedDB (клиент) |
| Доступность | Работает без интернета |
| Аутентификация | JWT (access + refresh) |
| Уведомления | Web Push API |
